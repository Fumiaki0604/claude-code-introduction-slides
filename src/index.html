    準備

    Before Getting Started

    必要なもの：

    • Anthropic アカウントの作成
    • Node.js のインストール (v16以上)
    • Git のインストール
    • コードエディター (VS Code 推奨)

    推奨環境：

    • macOS / Linux / Windows (WSL2)
    • ターミナルアクセス
    • インターネット接続

    インストール

    Installation

    # グローバルインストール
npm install -g @anthropic-ai/claude-code

# バージョン確認
claude-code --version
    # ワンラインインストール
curl -fsSL https://claude.ai/install.sh | sh

# パスの設定
export PATH="$HOME/.claude-code/bin:$PATH"

    基本的な使い方

    Basic Usage

    # Claude Code を起動
claude-code

# プロジェクトで初期化
claude-code init

# ヘルプを表示
claude-code --help

    • ファイル操作と編集
